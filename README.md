## Sample FastHttp Rest Server

Tutorial : https://medium.com/@muhammet.arslan/how-to-create-a-rest-server-in-golang-with-fasthttp-and-wire-f2a127a0d4fa

## Install

- Pull the repo
- run `glide install` to fetch dependencies
- `cd cmd/`
- `go run .`
